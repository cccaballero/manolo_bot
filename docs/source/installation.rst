Installation
============

Prerequisites
-------------

* Python 3.12 or higher.
* `uv` (recommended) or `pip`.

Installing from source
----------------------

Clone the repository:

.. code-block:: shell

   git clone https://github.com/cccaballero/manolo_bot.git
   cd manolo_bot

Install dependencies using `uv`:

.. code-block:: shell

   uv sync

Or using `pip`:

.. code-block:: shell

   pip install .
